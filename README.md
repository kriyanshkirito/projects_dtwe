# daily-project-till-the-world-exist

## Day 01 – Advanced Web Scraping Project

**Objective**: Build a flexible and ethical web scraper capable of extracting structured data (text, images, tables, links, etc.) from almost any website.

**Main topics covered**:
- Static & dynamic page scraping
- BeautifulSoup + Requests
- Selenium for JavaScript-rendered content
- Data cleaning & structured output (JSON, CSV)
- Rate limiting, user-agent rotation, error handling

## Day 02 – Crop Recommendation System

**Objective**: Predict the most suitable crop for a given area using soil and weather features with machine learning.

**Main topics covered**:
- Exploratory Data Analysis (EDA)
- Feature engineering & visualization
- Multi-class classification (Random Forest, etc.)
- Model evaluation (accuracy, precision, recall, confusion matrix)
- Practical agriculture use-case
